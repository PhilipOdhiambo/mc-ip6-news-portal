News Portal API

Welcome to news-portal APIThis page documents Ways to consume the API

Base href is: https://mc-news-portal.herokuapp.com

  • To display list of users:
    /api/employees
  • Add a new user:
    POST
    /api/departments/:departmentId/users/new
  • Get specific user:
    /api/users/:id
  • To display list of departments:
    /api/departments
  • To display one department:
    /api/departments/:id

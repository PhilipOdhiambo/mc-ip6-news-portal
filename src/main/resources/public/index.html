News Portal API

Welcome to news-portal API. This page documents Ways to consume the API

You can iteract with the GET routes in a new browser tab

Use postman to for POST routes

  • To display list of Employees:
    https://mc-news-portal.herokuapp.com/api/employees
  • Add a new user:
    POST - use postman
    https://mc-news-portal.herokuapp.com/api/departments/:departmentId/users/new
  • Get specific Employee:

    https://mc-news-portal.herokuapp.com/api/employees/:id

    Example: https://mc-news-portal.herokuapp.com/api/employees/1
  • To display list of departments:
    https://mc-news-portal.herokuapp.com/api/departments
  • To display one department:

    https://mc-news-portal.herokuapp.com/api/departments/:id

    Example: https://mc-news-portal.herokuapp.com/api/departments/1
